1. Overview & Architecture

The NSE Intraday Scanner is a fully local web application. It fetches real-time OHLCV data from Upstox, computes ORB, RSI, VWAP and ATR, generates confidence-scored trade signals with a 6-gate readiness checklist, and uses Claude AI to write plain-English trade plans. Nothing leaves your machine except API calls to Upstox and Anthropic.

🐍
upstox_proxy.py
Python server on port 8080. Proxies Upstox, Anthropic, NSE. Serves the app and README. Run it every morning.
🌐
nse_scanner.html
The full app — served at http://localhost:8080. Works in Chrome, Edge, Firefox on PC and mobile.
📄
nse_history.json
Your trade history database. Plain JSON on your PC. Link it once in the History tab.
📡
Upstox API
Real-time LTP, intraday 1-min candles, 30 days daily OHLCV. Requires daily token.
🤖
Anthropic API
AI market summary and per-stock trade plans. Requires a personal API key (one-time setup).
🏛
NSE India API
Public corporate actions data. Auto-fetched to warn about results, dividends, splits in next 7 days.

2. Daily Setup

Generating your Upstox Token

The Upstox access token expires at midnight IST every day. Run this once each morning:

cd C:\Abhi\Learning\stock_preditor python GetUpstoxToken.ps1
1
Run the script
Browser opens the Upstox login page automatically.
2
Log in with OTP
Use your Upstox mobile number and OTP. Browser redirects to https://localhost/?code=...
3
Copy redirect URL
Copy the full URL from the address bar and paste into PowerShell when prompted.
4
Token copied to clipboard
Script prints the token, copies it to clipboard, and saves to upstox_token.txt.

Starting the Proxy Server

cd C:\Abhi\Learning\stock_preditor python upstox_proxy.py

The proxy starts and automatically opens http://localhost:8080 in your default browser. It also prints your local network IP for mobile access. Keep this PowerShell window open the entire time you are trading.

⚠ Always use http://localhost:8080Never open by double-clicking the HTML file. Opening as file:// prevents Chrome and Firefox from making API calls due to browser security policies.

Accessing from Mobile (same WiFi)

The proxy listens on all network interfaces, so any device on the same WiFi network can reach it.

1
Find your PC's IP
The proxy prints it at startup, e.g. http://192.168.1.45:8080. Or run ipconfig in PowerShell and look for IPv4 Address.
2
Allow port 8080 through Windows Firewall (one-time)
Run PowerShell as Administrator:
New-NetFirewallRule -DisplayName "NSE Scanner" -Direction Inbound -Protocol TCP -LocalPort 8080 -Action Allow
3
Open on phone
Connect phone to same WiFi. Open Chrome/Safari and go to http://192.168.1.45:8080.
ℹ For access outside home WiFiInstall Tailscale (free for personal use) on your PC and phone. It creates a private VPN between your devices so the scanner works anywhere — mobile data, office, anywhere — with zero router configuration.

5. Signal Lock & Reversal Detection

The scanner remembers every signal for each stock across all scans in the same trading session. This lets you see whether a signal is stable or erratic.

How it works

  • Session lock — the very first signal for each stock each day is locked (🔒). It appears below the stock name on every subsequent scan.
  • Signal history dots — small coloured circles (B=Buy green, S=Sell red, W=Watch amber) show the sequence of signals across scans. The current scan has a black ring. Oldest → newest, left to right.
  • Reversal detection — if the current signal differs from the locked signal (and both are directional, not WATCH), a red ⚡ REVERSED badge appears on the row, and a red banner at the top of the scanner lists all reversed stocks.

Advisory only

The signal lock is advisory — the scanner shows you both the locked signal and the current signal. You decide. The entry price, target and stop loss are always based on the current scan. The lock is there to inform, not override.

Session reset

Signal history resets automatically at midnight every day so each trading day starts completely fresh.

💡 Practical ruleIf you see more than one reversal in a stock's history dots (e.g. S → B → S), that stock is ranging sideways. The ORB breakout direction is unclear. Skip it for the entire session and check again the next day.

6. Candle Confirmation

A signal appearing on one candle is not enough to enter. The scanner requires the last 3 consecutive one-minute candles to all close on the correct side of VWAP before marking a signal as confirmed.

StateBadge shownWhat to do
3/3 candles agree✓ 3/3 confirmedGate 3 is green — proceed with other gates
1 or 2 candles agree⚡ 2/3 candlesGate 3 is red — wait one more candle (1 minute), rescan

Unconfirmed signals also receive a −20% confidence penalty to reflect the higher risk. If a stock shows 78% confidence but is unconfirmed, it drops to 58% — pushing it into the half-size zone automatically.

⚠ Why this mattersAt 9:30 AM prices are volatile. A single candle closing above ORB High can be a spike that reverses immediately. Three consecutive candles closing above ORB High means buyers are committed. This one rule eliminates most false breakout entries.

8. Technical Indicators

ORB — Opening Range Breakout

Calculated from the first 15 one-minute candles (9:15–9:30 AM IST). Highest high = ORB High. Lowest low = ORB Low.

  • Price closes above ORB High + volume spike → BUY signal
  • Price closes below ORB Low + volume spike → SELL signal
  • Price inside range → WATCH
⚠ Never enter before 9:30 AM. The ORB must be fully formed first.

VWAP — Volume Weighted Average Price

Calculated from all intraday 1-minute candles, reset at 9:15 AM each day. Price above VWAP = buyers in control. Price below VWAP = sellers in control. Also used for candle confirmation (all recent candles must close on the correct side).

RSI (14) — Relative Strength Index

Calculated from the last 14 daily closes, not intraday candles.

RSIZoneOn BUYOn SELL
0–30Oversold✅ Strong bounce❌ Risky to short
30–60Neutral✅ Ideal✅ Ideal
60–70Strong⚠ Caution✅ Good short
70–100Overbought❌ Avoid✅ Strong short

ATR — Average True Range

14-day average daily range. Sets stop loss (entry ± 0.3–0.4 × ATR) and target (entry ± 1.5–2.0 × ATR). Ideal: 0.8–2.5% of price.

9. Corporate Events Warning

Fetched automatically from NSE India's public API. Any stock with an event in the next 7 days is flagged in the quick-add pills, selected stock tags, a yellow warning banner, the scan table, and the detail panel.

📋ResultsHigh impact. Stock can gap 5–15%. ORB signals unreliable. Skip on results day.
✂️Split / BonusHigh impact on ex-date. Price adjusts by ratio — all historical levels become meaningless.
💰DividendLow impact. Price adjusts by dividend amount on ex-date. Trade with awareness.
🏛AGMMinimal direct impact. Monitor news on AGM day.

10. History Tab

Track predictions vs actual outcomes to measure scanner accuracy over time.

Tab layout (top to bottom)

  1. Stats bar — Predictions / Target hit / SL / Win rate / Avg P&L
  2. 📁 JSON File Storage (collapsible)
  3. 📊 Accuracy by Category (collapsible)
  4. 🔍 Filters & Sort (collapsible)
  5. Export / Import / Clear buttons — always visible
  6. Bulk action bar — appears when rows are selected
  7. History table — always visible at bottom

Persisting data across browsers

Expand 📁 JSON File Storage → type your path → click Set path & load. Example: C:\Abhi\Learning\stock_preditor\nse_history.json. Every save writes to this file automatically. Works in Chrome, Edge, Firefox identically.

Recording outcomes

Fill in Actual entry and Actual exit after the session. Outcome and P&L are calculated automatically.

Bulk delete

Checkbox on each row. Header checkbox selects all visible. Bulk action bar appears with delete count. ✕ button on row hover deletes single row.

Export & Import CSV

Export downloads nse_history_YYYY-MM-DD.csv. Import merges from a previously exported CSV — skips exact duplicates (same stock + same date).

11. Trading Best Practices

Position sizing

<55%🔴 Not readyNo trade—

Hard rules

  • Never enter before 9:45 AM
  • Never trade a reversed signal (⚡ badge)
  • Never trade a WATCH signal
  • Never risk more than 1% of total capital on a single trade
  • Max 3 open positions at any time
  • Stop trading for the day after 2 consecutive losses
  • Place stop loss order immediately after entry — before anything else
  • Use limit orders at Entry price, never market orders
  • Exit all positions by 3:15 PM
  • Never trade on results day or split ex-date
